## HTML

### 基本语法

1. 标签通常成对出现，开放标签和闭合标签

2. `<html>`描述网页，`<body>`为可见内容，`<h1>`为标题，`<p>`为段落，`<a>`为链接，`<img>`为图像（图像其他是由属性提供），`<br />`是没有关闭标签的空元素，定义换行

3. 标题包括`<h1>`到`<h6>`

4. 元素数量取决于标签对数

5. 属性示例：

     ```html
   <a href="http://www.w3school.com.cn">This is a link</a>
     ```

   属性写在标签头，以键值对的形式。

   此处`This is a link`会变成超链接形式。

   属性值内含有`""`，则应用`''`括住属性。

   通用属性：`class`：元素类名

   ​                    `id`：元素id

   ​                     `style`：元素样式

   ​                       `title`：元素额外信息

6. 水平线创建：`<hr />`无标签尾

7. 注释：`<!-- comment -->`

8. 折行：`<p>a<br />b</p>`

9. 换行只能换一行，空格无效
